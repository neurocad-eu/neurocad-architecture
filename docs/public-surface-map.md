# Public Surface Map

## Public repositories by function

- `neurocad-api-docs` for contracts, schemas, and integration guidance
- `neurocad-examples` for runnable reference clients and webhook patterns
- `neurocad-benchmarks` for release-packaged benchmark artifacts
- `neurocad-architecture` for system boundary and diligence-facing architecture notes

## Public interface categories

- authentication and token issuance
- project and job orchestration contracts
- artifact resolution and callback flows
- benchmark packaging and metadata
- public architecture and trust-boundary documentation

## Private implementation categories

- geometry kernel internals
- private job orchestration implementation
- internal data generation and evaluation systems
- operational infrastructure and deployment topology
